
This is an example based on Altium 14 output.  Includes
an IPC-D-356A output (*.ipc) and PnP CSV files.

Original output (standard RS274-X) is in 
  HeadPower
  HeadControl2
  DriverBoard
directories, including NC drill (.TXT), PnP (.csv)
and .ipc.

Selected files manually modified with RS274-X2 attributes, 
in x2 sub directories.
